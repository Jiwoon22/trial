XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB (STCOM) 제품은 컴퓨팅 제품군 전문 용통사인 (주) STCOM에서 출시를 했습니다. (주)에스티컴퓨터는 1998년 2월 창립되었으며, 이듬해인 1999년 ASUSTEK 한국 공식 유통사가 되었으며 이후 국내 컴퓨팅 분야에서 ASUSTEK과 함께 성장해왔습니다. (주) STCOM은 ASUS 메인보드, PALIT 그래픽카드, 실버스톤 공식 수입원입니다.


XFX 라데온시리즈에는 MERC(머큐리), QICK(퀵), SWFT(스위프트)시리즈가 있습니다. 그중에 이번 사용기 제품은 SWFT(스위프트)시리즈 제품으로 고해상도 및 그래픽 효과로 최신 게임 플레이의 최적화된 성능을 제공합니다.

☞ 2023년 05월 20일 다나와 기준으로 XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB 제품의 가격으로 약 43만원에 판매를 하고있습니다. 2021년 08월 20일 다나와 기준으로 XFX 라데온 RX 6700XT제품이 약 70후반에서 110만원까지 그래픽카드가격이 금값이였습니다. 아니 돈이 있어도 구입을 하기 힘든시기가 있었지만 ~ 지금은 가격이 많이 하락이 되어 구입하기 좋은 시기입니다. 저렴할때 필요하시면 구입하시면 좋습니다. XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB 제품은 공식 유통사인 (주)STCOM에서 보증을 하며, 3년 무상보증 AS서비스를 제공하여 안심하고 구입을 하셔도 됩니다.

2. 개봉기


▲ AMD는 2020년 12월 "라데온 RX 6000시리즈"를 출시했습니다. 그러나 가상화페로 소비자가 구입할수있는 그래픽카드는 생각보다 없었습니다. 시장에서 구입을 할수 없었으니까요 그리고 2022년 12월 " 라데온 RX7000시리즈:를 출시를 했습니다. 그러나 지금 출시한 제품은 7900XTX으로 백육만원정도 하고있어, 일반유저에게는 부담이 되는 가격입니다. AMD 그래픽카드를 좋아 하는 저에게도 큰 부담입니다. 그런 유저들을 생각해서 6000시리즈가 계속 출시를 하고있습니다. AMD는 소비자의 반영을 생각해서 계속 신제품을 출시를 할것이라 생각합니다. 이번에 새로 나온 XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB 제품도 비슷한 유저의 바램으로 출시를 한것이라 생각합니다. 그중에 " 라데온 6700 "는 이전 세대 대비 개선된 아키텍처와 향상된 GPU와 메모리를 기반으로 뛰어난 그래픽 성능은 물론 전력 면에서도 대단히 좋은 모습을 보여주며서, 가성비 좋은 제품으로 출시를 해서 더욱 기다리는 게이머의 마음에 불을 활,활,활 타오르게 했습니다. 신제품 XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB 그래픽카드는 최신의 AMD 7nm 공정과 차세대 그래픽 아키텍쳐인 AMD RDNA2 아키텍처 기반으로 AMD 인피니티 캐시, DirectX 레이트레이싱 (DXR) 등 최신 기술이 접목이로 많은 부분에서 더 좋아진 모습으로 변경이 되었으며, Navi 22 코어를 기반으로 만들어졌으며, 36개의 컴퓨트 유닛과 2304개의 스트림 프로세서로 되어 있으며, 베이스클럭: 1941MHz, 부스트클럭: 2425MHz이며, 80MB 인피니티 캐시가 탑재된 것은 물론, 기본 메모리 용량도 10GB(GDDR6의 16000MHz (160 bit))로 대단히 높은 초고대역폭의 메모리로 보다 생생한 시각 효과와 고해상도 게이밍 성능을 발휘합니다. XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB 그래픽카드는 XFX 사만의 차원이 다른 최고 수준의 고급스러운 디자인과 블랙 컬러로 심플함까지 더해 성능뿐만 아니라 외형적으로도 최고의 완성도를 자랑하는 제품입니다.


▲ ASUS TUF Gaming A620M-PLUS + AMD 라이젠5-5세대 7600X (라파엘) + XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB에서 테스트를 했습니다.

4. 총평


☞ 지금까지 XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB 그래픽카드를 사용 했습니다. 제품을 잘 만들었다고 생각합니다. 드라이버 안정성도 점점 좋아지고있고, 소음, 발열, 마감등 좋은 제품입니다. 그리고 제일 좋은점은 가성비가 좋은 제품이라 대 만족합니다. 1080p 게임에서는 새로운 표준 제품이라고 할수있습니다.


XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB는 이전 세대 대비 향상된 아키텍처에 인피니티 캐시와 고용량 메모리 등을 탑재해 경쟁 제품과 비교해서 손색이 없는 모습을 보여주는 것을 확인할 수 있었습니다. 벤치 점수를 보면 경쟁회사의 제품보다 높은점수도 있지만, 그렇지 못한점수들이 더 많았지만, 그래도 전세대의 제품보다는 많은 발전의 모습을 보여주었습니다.


채굴 시장이 정점이였던 2021년 여름쯤 라데온제품은 정말 비싼 제품이였습니다. 아니 시장에서 제품을 구할수 없었습니다. 그러다 2022년에 LHR버전의 그래픽카드가 출시를 하고 2023년에 오면서 여러 정부의 코인 규제 소식 및 재제가 들어 오면서 코인의 내리막을 걷기 시작하면서 그래픽카드의 가격이 하락이 되고있습니다. 그럼으로 다시 여러가지의 그래픽카드의 제품을 출시를 하고있습니다. 게임을 좋아하는 유저에게는 방가운 소식이지만 업체에게는 좋은 소식은 아니지요 그렇지만 다른 한쪽에서는 중고제품을 재포장해서 제품을 출시하는경우도 있습니다. 그러나 (주) STCOM에서 수입하는 XFX 라데온 제품은 그렇지 않습니다. 새제품입니다. 돈 한푼두푼 아니지 않습니까 믿고 구입을 할수있는 (주) STCOM에서 수입하는 XFX 라데온 RX 6700 SWFT 309 CORE D6 10GB을 구입하시면 믿고 구입하실수있습니다.
